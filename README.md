# Jaxon's Numbers Game

Display name: **Numbers R' Everywhere**

A blue, blocky, Minecraft-inspired number playground with heavy floating numbers, blinking numbers, cursor-follow number sparks, speech, a calculator, and seven number games.

## What is inside

- Big calculator that can talk out loud
- Keyboard input for typing number expressions
- QR code card for opening the GitHub Pages app on iPhone or tablet
- Floating number background with cursor-follow behavior
- Blinking number buttons
- Cursor-follow number sparkles
- Stars saved in the browser
- Seven games:
  - Prime Pop
  - Block Arrays
  - Number Duel
  - Pattern Finder
  - Target Builder
  - Speed Add
  - Memory Match

## Quick Start

Open `index.html` in a browser.
